### Описание программы:

Таблица в формате Single Page Application (SPA), которая будет отображать данные из базы данных и обеспечивать пользователю функциональность для сортировки, фильтрации и пагинации данных. Таблица должна работать без перезагрузки страницы, предоставляя пользователю удобный и интерактивный интерфейс для взаимодействия с данными.

 

### Задача: 

**Требования к таблице:**


1. **Структура таблицы:** 
   - Таблица содержит 4 колонки: 
     - Дата
     - Название
     - Количество
     - Расстояние
2. **Функциональность таблицы:** 
   - **Сортировка:** 
     - Таблица имеет возможность сортировки по всем полям, кроме даты.
   - **Фильтрация:** 
     - Таблица содержит два выпадающих списка и текстовое поле для фильтрации данных: 
       - Выбор колонки для фильтрации.
       - Выбор условия фильтрации (равно, содержит, для чисел больше, меньше).
       - Поле для ввода значения для фильтрации.
   - **Пагинация:** 
     - Таблица содержит функциональность для пагинации данных.
     - Пагинация реализована на стороне сервера.
3. **Автоматическое заполнение данных:** 
   - При старте приложения база данных автоматически заполняется случайными значениями для демонстрации функциональности.
4. **Контейнеризация и развертывание:** 
   - Проект "обёрнут" в Docker для облегчения развёртывания и обеспечения переносимости.
   - В проекте настроен Nginx для проксирования общения между фронтендом и бекендом.


**Технологии и инструменты:**

- **Backend:** 
  - Python 3.11
  - Django REST Framework для создания серверного API
  - PostgreSQL для хранения данных
  - Django ORM для работы с базой данных
- **Frontend**: 
  - Vue.js для создания интерактивного интерфейса
  - Axios для выполнения HTTP-запросов к серверному API
  - CSS библиотеки для стилизации (например, Bootstrap)
- **Проксирование:** 
  - Nginx для проксирования запросов между фронтендом и бекендом
